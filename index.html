GUIDE KONEKSI

Cara Koneksi menggunakan PowerTunnel:

  1. Install PowerTunnel dari Google Play Store.
  2. Buka PowerTunnel, lalu klik menu (tiga garis).
  3. Pilih "Settings", lalu "Tunnel Configuration".
  4. Di bagian *Host*, masukkan **gt.growrushps.xyz**.
  5. Aktifkan PowerTunnel dan buka Growtopia.

Cara Koneksi menggunakan Virtual Host:

  1. Pastikan aplikasi Growtopia asli sudah ter-uninstall.
  2. Download dan install Growtopia MOD dari link di atas.
  3. Download file Virtual Host (di tombol Virtual Host).
  4. Ekstrak file zip/rar tersebut ke folder Growtopia Anda.
  5. Jalankan Growtopia MOD.

Cara Koneksi menggunakan PC Client:

  1. Download PC Client melalui tombol di atas.
  2. Ekstrak file ZIP di folder mana pun.
  3. Jalankan file `.exe` yang sudah di-ekstrak.
  4. Client akan otomatis melakukan koneksi ke server.
  5. Jika error, pastikan Anda memiliki `.NET Framework` terbaru.

Hubungi admin jika koneksi PC gagal setelah mencoba langkah ini.
